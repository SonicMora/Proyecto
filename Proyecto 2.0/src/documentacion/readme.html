Proyecto Final Algoritmos y Programacion II

Para este proyecto decidí que sería mas eficiente tener

*Una lista enlazada simple para el manejo de los enemigos

*Un arbol binario de busqueda para el manejo de los usuarios

*3 excepciones propias

*Una en caso de que el arbol este vacio y el usuario

quiera realizar acciones sobre este

*Una en caso de que un nuevo usuario quiera registrarse

con un nombre que ya haya sido registrado anteriormente

*Finalmente una en caso de que el usuario quiera buscar

un puntaje que no existe

*Hilos para el movimiento de los enemigos, el movimiento

de el usuario, el avance en pantalla de el puntaje del mismo

y para los disparos

Proyecto Presentado por Victor Mora
